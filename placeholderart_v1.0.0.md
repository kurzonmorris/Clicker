# Project Arasmas — Placeholder Art Tracker

> **Purpose:** Complete list of ALL placeholder art assets in the game. Every entry here is AI-generated and must eventually be replaced by artist-produced final art.
> **Last Updated:** 2026-04-03
> **Status:** Template created — will be populated as development progresses and assets are identified

---

## How to Use This File

- Every AI-generated art asset used in the game gets an entry here
- When an artist replaces an asset, mark it `[x]` and note the date and artist
- This file is the single source of truth for art replacement progress
- Categories will expand as new systems and content are added

---

## Status Key

- `[ ]` = Placeholder (AI-generated, needs replacement)
- `[~]` = In progress (artist is working on replacement)
- `[x]` = Replaced (final art in place)
